# AGENTS.md

## Project overview
개인 재무 관리 AI 서비스

## Setup
- Install: `npm install`
- Build: `npm run build`
- Test: `npm test`

## Code style
- TypeScript strict mode
- ESLint + Prettier 사용
- 함수형 패턴 선호
- Always add JSDoc comments for public APIs

## Testing instructions
- PR 전에 반드시 `npm test` 실행
- 새 기능은 테스트 필수

## PR guidelines
- 제목 형식: `[module] Description`
- 커밋 메시지는 conventional commits 따름
